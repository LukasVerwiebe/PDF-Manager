# PDF-Manager
Datei-Manager für PDF-Dateien mit dem Dateien aneinandergefügt, ausgelesen und aufgesplittet werden können.
Das Programm wurde mit Python erstellt, dabei wurden die folgenden Bibliotheken verwendet:
- PyQt5
- PyPDF2
